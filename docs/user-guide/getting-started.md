# Getting Started

Welcome to Necroverse! This guide will help you get started with preserving and running legacy file formats.

## What is Necroverse?

Necroverse is a platform for preserving and running legacy file formats in modern browsers. It supports:

- **SWF** (Shockwave Flash) files
- **JAR** (Java Archive) files
- **XAP** (Silverlight Application) files
- **DCR** (Shockwave Director) files
- **EXE**, **DLL**, **OCX** (Windows executables)

## Quick Start

### 1. Upload a File

1. Go to **NecroDev** (the upload interface)
2. Click **Upload File**
3. Select a supported file format (SWF, JAR, XAP, DCR, etc.)
4. Wait for conversion to complete

### 2. Play a File

1. Go to **NecroPlay** (the player interface)
2. Enter the file ID or browse your files
3. Click **Play** to start playback
4. Use playback controls to navigate

### 3. Share a File

1. Open a file in **NecroPlay**
2. Click **Share** button
3. Copy the share link or embed code
4. Share with others

## Features

### Playback Controls

- **Play/Pause**: Start or pause playback
- **Stop**: Stop playback and reset to beginning
- **Frame Navigation**: Use arrow keys or buttons to navigate frames
- **Speed Control**: Adjust playback speed (0.25x to 2x)
- **Timeline Scrubber**: Drag to jump to any frame

### Keyboard Shortcuts

- **Space**: Play/Pause
- **Arrow Left**: Previous frame
- **Arrow Right**: Next frame
- **F**: Toggle fullscreen
- **Esc**: Exit fullscreen

### Visual Enhancements

- **CRT Scanlines**: Toggle CRT scanline effect
- **Pixel Perfect**: Enable pixel-perfect rendering
- **Zoom**: Zoom in/out/reset
- **Screenshot**: Capture current frame

### Performance Monitoring

- **FPS**: Current frames per second
- **Frame Time**: Time per frame
- **Memory Usage**: Current memory usage
- **Render Time**: Time spent rendering

## File Format Support

### SWF (Shockwave Flash)

- Supports ActionScript v1.0
- Audio playback (MP3, ADPCM)
- Timeline animation
- Interactive buttons and sprites

### JAR (Java Archive)

- Java bytecode execution
- Exception handling
- Method invocation
- Field access

### XAP (Silverlight Application)

- XAML parsing and rendering
- Brushes and transforms
- Nested elements
- Attached properties

### DCR (Shockwave Director)

- Lingo script execution
- Sprite channel management
- Frame scripts
- Timeline control
